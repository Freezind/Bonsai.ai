Bonsai · Architecture

Fed by your life, tending it back. One rule holds the whole stack together: the model emits structure, never code — everything it says is transpiled on-device, through a frozen widget pool, into a fully native app.

iPhone · Flutter app 100% native

🌱 Onboarding

Native Dart seed flow. Fixed 2-round AI conversation → PARA classification (Project / Area). Scripted fallback when offline.

📔 Home digest

Native Dart periodic note: the gardener's weekly report + per-goal highlights with grown trees.

🗂 PARA shell

GoRouter 5-tab shell (Home·P·A·R·A), sub-pages capped at depth ≤ 3 — enforced on device.

🎨 Goal dashboards · rfw

DSL text → Flutter rfw transpiles it into a real native widget tree. Parse failure keeps the prior screen — never blank.

🔒 Frozen widget pool

bonsai.widgets — 70+ whitelisted components + Fresh Matcha tokens. The only vocabulary the model may speak.

💾 ScreenStore

Memory + disk DSL cache, in-flight dedupe. Generated once → renders instantly forever.

HTTP · LAN / USB-tether · POST /generate · rfw DSL only
Mac · local bridge serve.py · zero deps

💬 converse / conclude

Onboarding turns: one AI follow-up per round; strict-JSON classification + closing copy. In-memory turn cache → retries are idempotent.

🧩 intent → DSL

Composes screens from the pool per the system prompt. leaf screens get intent-links stripped mechanically.

🗃 Bridge cache

cache.json keyed sha1(system_prompt + intent) + in-flight dedupe — each intent is generated once, ever.

🤖 headless claude -p

Local Claude login, no API key. The model is a supplier behind the bridge — swappable without touching the app.

🔐

The capability lock

The model can only compose whitelisted structure — never executable code — and rendering never leaves the device. The pool is kept in three-way sync: lib/rfw_pool/local_widgets.dart (Dart implementations) · bridge/system_prompt.txt (what the model knows) · lib/ds/aurora_tokens.dart (Fresh Matcha design tokens). This is what makes an AI-composed interface App-Store-shippable.
