Bonsai Design System

A token-first, self-growing UI kit. The reference guideline is Material tokens + MUI components (cross-platform). Source of truth: styles.css. Toggle Dev Mode to inspect any component; use Copy as prompt to build it in any language (Flutter first). Switch the theme top-right — only :root tokens change, every component follows.

Colors

Semantic roles — every fill is paired with its --on-* foreground (WCAG-checked by math). Neutrals are tinted warm toward paper; status colors are storybook pops.

Typography scale

Baloo 2 display × Nunito body × Patrick Hand accent. ~1.25 modular scale, tracking tightened as size grows.

Spacing

One 4px-based scale used everywhere. No ad-hoc pixels.

Radius

Soft, generous rounding for the hand-drawn feel.

Elevation

Soft ink-tinted ambient shadows, plus the signature hard offset (--elev-pop) — the picture-book cut-out that collapses on press.

Mascot & brand

Bonsai’s mascot is an original hand-drawn sprout with a cheeky little face (幼齿但清新、贱萌). Watered with your context, it matures seed → sprout → bonsai → bloom → fruit, and reacts with moods (thirsty, wilting, sleeping, celebrating). Fully themeable (fills read --leaf/--pot/--bloom). See the full state sheet + the Lottie motion spec.
