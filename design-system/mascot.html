The Bonsai mascot, redrawn 1:1 from the shipped Lottie files — thin clean ink outline, flat fills only, minimal face (dot eyes · tiny smile · blush 腮红). Colors are pinned to the Lottie hexes via the --m-* tokens (--m-leaf #3FA34D, --m-pot #E8703A, --m-ink #33302B…), so static SVG and animation never drift. Triggers, loops and the state machine live in motion-lottie.md.
